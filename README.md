keadatabase-front
=================

Setup
-----
You will need to have Node >= 4 installed (and npm).
Then run `npm install`.

Running
-------
To run on your local machine at <http://localhost:3000/> run:
`npm start`

Building
--------
To build the app for production run:
`npm run build`
